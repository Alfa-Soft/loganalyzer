phpLogCon - Getting Started

Getting started with phpLogCon is just a few simple steps:

  • make sure your system meets the minimum requirements
  • under windows, run the supplied setup program - under other environments, extract the archive to a proper location
  • point your browser to the root of the phpLogCon
  • configure phpLogCon via the browser-based dialog that pops up
